# Sabores del Mundo

Sitio web estático para un restaurante multicultural que ofrece comida mexicana, italiana, japonesa y china.  
El proyecto fue desarrollado como práctica académica en equipo para aplicar estructura semántica en HTML y estilos básicos con CSS.

## Descripción

Este proyecto consiste en un sitio web de un restaurante ficticio llamado **Sabores del Mundo**, el cual presenta distintos menús internacionales y una página de contacto.  
El objetivo principal es aplicar buenas prácticas de desarrollo web, como el uso de etiquetas semánticas en HTML, metodología BEM para CSS y diseño de layouts utilizando Flexbox.

El proyecto no busca ser una aplicación funcional completa, sino demostrar el correcto uso de estructura, orden y estilos básicos.

## Tecnologías Utilizadas

- HTML5  
- CSS3  
- Normalize.css  

## Requisitos Previos

- Navegador web moderno (Google Chrome, Firefox, Edge, Safari)
- Editor de código (recomendado: Visual Studio Code)

## Instalación

1. https://code.visualstudio.com/
2. Abre el archivo `index.html` en tu navegador web.

No se requieren dependencias adicionales ni instalación de librerías.

## Uso

1. Abre el archivo `index.html`.
2. Navega entre las distintas secciones del sitio usando el menú de navegación.
3. Explora los diferentes menús de comida internacional y la página de contacto.

## Características

- Página de inicio con sección Hero informativa.
- Menús de comida mexicana, italiana, japonesa y china.
- Página de contacto con formulario.
- Uso de estructura semántica (`header`, `main`, `section`, `article`, `footer`).
- Estilos básicos con CSS.
- Implementación de Flexbox para diseño de layouts.
- Uso de metodología BEM para nombrado de clases.
- Proyecto organizado mediante estructura de carpetas.
